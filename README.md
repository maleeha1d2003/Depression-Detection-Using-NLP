# Depression Detection Using NLP

A Machine Learning and Natural Language Processing (NLP) project designed to identify signs of depression and mental health risk from social media posts. The system analyzes textual content, extracts linguistic and emotional features, and predicts whether a post indicates depressive tendencies.

## Project Overview

Mental health issues such as depression have become increasingly prevalent in the digital era. Social media platforms provide valuable insights into users' emotions and mental well-being through their posts and interactions.

This project applies Data Science, NLP, Sentiment Analysis, and Machine Learning techniques to classify social media posts as Depressed or Not Depressed. The system includes a trained machine learning model, Flask backend API, and Flutter mobile application for real-time predictions.

## Features
